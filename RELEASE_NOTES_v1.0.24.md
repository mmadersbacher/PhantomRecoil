# Phantom Recoil v1.0.24

## Overview
UI direction shift to a classic industrial/brutalist style with strong animation accents and no neon/rainbow aesthetic.

## Design Upgrade
- Replaced the previous shiny/neon atmosphere with a darker tactical palette.
- Introduced bold type system (`Bebas Neue`, `Barlow Condensed`, `IBM Plex Mono`) for a stronger identity.
- Reworked cards, tabs, controls, and status elements with hard edges, stripe motifs, and utility-first contrast.
- Added purposeful motion language (scanlines, active-stripe sweeps, status/favorite pulses) without ambient glow effects.

## Cleanup
- Removed cursor spotlight ambient layer from the frontend.
- Removed 3D pointer tilt behavior on operator cards for a cleaner classic feel.
