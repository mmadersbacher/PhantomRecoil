# Phantom Recoil v1.0.22

## Overview
Design-focused release with stronger motion language, richer atmosphere, and more interactive UI feedback.

## Design Upgrade
- Added animated ambient background orbs for a more alive visual layer.
- Added cursor-driven spotlight glow across the main area.
- Added interactive 3D tilt on operator cards (pointer-fine devices).
- Added shimmer sweep effect on weapon buttons.
- Added animated topbar edge scanline.
- Added staggered card reveal timing during grid rendering.

## UX / Accessibility
- Motion remains guarded with `prefers-reduced-motion` fallback.
- Mobile/responsive behavior from the existing layout is preserved.
